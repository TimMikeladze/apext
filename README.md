# APEXT 
A simple CLI tool to output all your Next.js API routes in your console.

### ⚠️ For now it is macOS only ⚠️

# Install

```bash
brew install lifespikes/apext/apext
```

# How to use

Place yourself on your Next.js root directory and run `apext list`

```bash
cd ./my-next-project
apext list
```
